Portfolio #YoProgramo

Desarrollo una aplicación web full stack que presenta: datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías y habilidades. Arquitectura distribuida y diseño de interfaz de usuario (frontend) que muestra la información, una base de datos que almacene los datos antes mencionados y las APIs necesarias para proveer a través de internet la información (backend).
